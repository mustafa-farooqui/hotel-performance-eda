# Hotel Booking Data Analysis (EDA)

## Project Overview
This project presents an Exploratory Data Analysis (EDA) on 135,000+ hotel booking records collected over a 3-month period.
The objective is to extract meaningful business insights related to revenue trends, guest behavior, cancellations, and overall hotel performance. The analysis focuses on transforming raw data into actionable insights for better decision-making in the hospitality domain.

## Business Problem
Hotels often face challenges such as:

- Fluctuating revenue patterns  
- High cancellation rates  
- Difficulty understanding customer behavior  
- Identifying seasonal demand trends  

This project aims to address these challenges using data analysis techniques.

## Key Insights
- Revenue trends and seasonal patterns were identified across the dataset  
- Comparison of performance across different hotel types and locations  
- Analysis of cancellation patterns and their impact on revenue  
- Understanding guest booking behavior and preferences  
- Identification of key drivers affecting hotel performance  

## Dataset Information
- 135,000+ hotel booking records  
- Data collected over a 3-month period  
- Multiple cities and hotel categories included  
- Features include bookings, cancellations, revenue, ratings, and guest-related attributes
